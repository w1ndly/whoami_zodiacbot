# ROADMAP

## Принцип разработки

Работаем только маленькими этапами.

Каждый этап проходит цикл:

1. Проектирование.
2. Небольшой участок кода.
3. Локальное тестирование.
4. Подключение к проекту.
5. Проверка работы.
6. При крупных изменениях:
   - Restart Railway;
   - Redeploy Railway.
7. Обновление документации.
8. Git commit.
