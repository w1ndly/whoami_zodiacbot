# CHANGELOG

## v0.3.0 — SQLite и стабилизация MVP

### Добавлено

* Добавлен `database.py` для работы с SQLite.
* Добавлен файл базы данных `bot.db`.
* Добавлена таблица `user_checks`.
* Добавлены функции чтения и увеличения счетчика проверок.
* Добавлен `storage.py` как слой хранения данных.

### Изменено

* Счетчик бесплатных проверок переведен с памяти Python на SQLite.
* `user_profile.py` теперь получает данные через `storage.py`.
* `bot.py` не зависит от SQLite напрямую.

### Исправлено

* Исправлено двойное списание проверки в сценарии пограничной даты без времени.
* Исправлена проблема устаревших inline-кнопок выбора города.
* Улучшена обработка ошибок при выборе города.
* Убраны дублирующиеся тексты и лишний неиспользуемый код.

### Статус

* Бот запускается.
* Лимиты сохраняются после перезапуска.
* SQLite подключен и протестирован.
* MVP стабилен.
